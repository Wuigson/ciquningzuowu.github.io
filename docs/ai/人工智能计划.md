# AI 学习路线（C 背景转型版）

## 📌 第一阶段：Python 速成（2 周）

### 学习重点

1. **语法差异对照**

   - 指针 → 引用（`int* p` → `lst = []`）
   - 结构体 → 类（`struct` → `class`）
   - 宏定义 → 常量（`#define` → 全大写变量）

2. **必须掌握的 Python 特性**
   - 上下文管理器（替代`fopen/fclose`）
   - 列表推导式（替代循环初始化）
   - 多返回值（替代指针参数）
   - ...

### 学习资源

- 推荐工具：VS Code + Python 插件、pycharm 等

## 🕸️ 第二阶段：爬虫与并发（2 周）

### 知识体系

1. **爬虫技术栈**

   - 基础库：Requests/BeautifulSoup
   - 框架：Scrapy
